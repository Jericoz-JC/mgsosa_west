# Find the Imposter word bank

## How to run each round

1. The Room Host privately sends the **group word** to every regular player.
2. Send the **Imposter word** to one randomly selected player. For a harder round, send only `IMPOSTER — no word` instead.
3. Players must not say either secret word. Each player gives one short clue per clue round.
4. After two clue rounds, allow 60 seconds of discussion and conduct the simultaneous Zoom-chat vote.
5. Use a different row for every new round. Do not announce the category before the vote.

The paired words are intentionally related. The Imposter can participate, but should notice that other clues do not fit their exact word.

## Host selection guidance

- Use Bank A for younger or mixed-age rooms.
- Use Bank B when everyone is comfortable with common New Testament knowledge.
- Use Bank C only when the room is familiar with Holy Qurbana vocabulary.
- Use Bank D for older participants or a second round.
- Avoid pairs that someone in the room is unlikely to recognize.
- If a player asks what their word means, privately choose another row for the entire room rather than explaining only to that player.
- Never display this document on the shared screen. Send each player only their own assigned word through private Zoom chat.
