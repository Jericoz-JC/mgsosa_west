# Gartic Phone — New Testament Prompt Bank

## Participant instructions

- Every opening sentence must use a New Testament person, place, event, object, or teaching.
- Keep prompts respectful, family-friendly, and possible to draw.
- Use one clear action instead of a long paragraph.
- Do not write private jokes about another participant.
- Use `first name + church` as your Gartic nickname.

The Room Host may read several examples before starting. Participants can write their own New Testament prompts or use one from this bank.

## Room Host opening script

“For your opening sentence, use a New Testament person, place, event, object, or teaching. Keep it respectful, family-friendly, and easy to draw. One clear action works best. For example: ‘Zacchaeus watches Jesus from the top of a tree.’ You may create your own sentence or use an idea from the prompt bank.”

## Album reveal and voting

1. Keep participants muted during writing and drawing turns.
2. Screen-share only the Gartic album reveal, never the room-host website console.
3. After every album is shown, ask participants to type the name of their favorite album in Zoom chat without sending it.
4. Count down, “3, 2, 1, send.”
5. Determine which website team the winning album's original participant belongs to.
6. In the room-host website, select **Finish reveal + vote**, then select that team under **Award 200**.
7. Submit only once; the website blocks a second award for the same room and rotation.

## Backup plan

If Gartic Phone is unavailable, open Skribbl and keep the same Zoom breakout and website participant code. Use New Testament people, places, and objects as the drawing-word list.
