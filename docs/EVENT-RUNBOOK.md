# July 11 online game-night runbook

## Before participants arrive

- 5:00 PM: Board and room admins join Zoom.
- Open Game Master, display, participant-test, and each room-admin view.
- Generate temporary room codes and paste the correct links into the room records.
- Test one buzz from two different devices.
- Confirm every room has a named host and backup.

## Fallbacks

- Site or Convex unavailable: use Zoom Raise Hand and record scores in the shared document.
- Gartic unavailable: use Skribbl and keep the same room code in the console.
- Room host missing: assign the floater before sending participants to breakouts.
- More than 48 participants: create additional Imposter and Gartic rooms before the first rotation.
