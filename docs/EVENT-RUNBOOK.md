# July 11 online game-night runbook

## Before participants arrive

- 5:00 PM: Board and room hosts join Zoom.
- Open Game Master, display, and a participant-test view.
- In **Game Master → Breakout rooms**, generate a participant code for each room.
- Create and privately send one volunteer link per room. Creating a replacement invalidates the old link.
- Open each volunteer link once and confirm it shows only the assigned room.
- Test one participant room join and one buzz from two different devices.
- Confirm every room has a named host and backup.

## Fallbacks

- Site or Convex unavailable: use Zoom Raise Hand and record scores in the shared document.
- Gartic unavailable: use Skribbl and keep the same participant room code.
- Room host missing: assign the floater before sending participants to breakouts.
- Volunteer link fails: replace it from the Game Master room console and send the new link privately.
- More than 48 participants: create additional Imposter and Gartic rooms before the first rotation.
